# Archived Docs

Folder ini berisi dokumentasi lama yang tidak lagi menjadi sumber kebenaran utama untuk repo `patroli_v2`.

Isi archive umumnya salah satu dari berikut:
- konten template asal
- guide untuk fitur yang tidak lagi aktif di produk ini
- landing page / marketing docs untuk static site lama
- referensi lama yang belum dibuang sepenuhnya demi histori

Rule praktis:
- kalau dokumentasi masih dipakai tim saat ini, letakkan di root `docs/`
- kalau hanya berguna sebagai referensi historis, simpan di `docs/archive/`
